### How Backup .bashrc
Backup your current .bashrc file:
```bash
cp ~/.bashrc ~/.bashrc.bak
```

Copy the skeleton .bashrc file over yours:
```bash
cp /etc/skel/.bashrc ~/
```

Afterwards, load the new one:
```bash
source ~/.bashrc
```
